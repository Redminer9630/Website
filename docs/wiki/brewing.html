Brauen

    • Netherwarze | Zutat fuer den Seltsamen Trank, der Grundlage für die meisten Traenke ist.

    Redstone | Mit diesem Grundstoff kann die Wirkungsdauer vieler Traenke verlaengert werden.

    Leuchtsteinstaub | Mit diesem Grundstoff kann die Staerke vieler Traenke erhoeht werden, was die Wirkungsdauer reduziert.

    Fermentiertes Spinnenauge | Mit diesem Grundstoff kann aus vielen positiven Traenken ein negativer Trank gebraut werden.

    Schwarzpulver | Mit diesem Grundstoff wird eine werfbare Version des Trankes gebraut. Allerdings wird dadurch die Wirkungsdauer reduziert.

    Drachenatem | Mit diesem Grundstoff wird eine verweilende Version des Wurftrankes gebraut.
