Brauen

    • Netherwarze | Zutat für den Seltsamen Trank, der Grundlage für die meisten Tränke ist.

    Redstone | Mit diesem Grundstoff kann die Wirkungsdauer vieler Tränke verlängert werden.

    Leuchtsteinstaub | Mit diesem Grundstoff kann die Stärke vieler Tränke erhöht werden, was die Wirkungsdauer reduziert.

    Fermentiertes Spinnenauge | Mit diesem Grundstoff kann aus vielen positiven Tränken ein negativer Trank gebraut werden.

    Schwarzpulver | Mit diesem Grundstoff wird eine werfbare Version des Trankes gebraut. Allerdings wird dadurch die Wirkungsdauer reduziert.

    Drachenatem | Mit diesem Grundstoff wird eine verweilende Version des Wurftrankes gebraut.
