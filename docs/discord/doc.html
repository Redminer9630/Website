Dokumentation Bot9630

Giveaways

Allgemeine Argumente:
titel: Definiert den Titel eines Giveaways. Wird bei jedem Giveaway Command benötigt.

/giveaway start

Startet ein Giveaway im jetztigen Kanal. Argumente:
dauer: Dauer des Giveaways. Kann in s|m|h|d angegeben werden.
gewinner: Ganze Zahl.
codes: Definiert die Gewinne, wird in DM so gesendet: "Du hast im Giveaway TITEL mit TEILNEHMER Teilnehmern gewonnen! Dein Gewinn: CODE".

/giveaway stop

Stopt den Timer vom Giveaway und wählt sofort Gewinner aus.

/giveaway list

Listet alle Giveaways auf. Grün = Giveaway Läuft, Blau = Giveaway Pausiert, Rot = Giveaway Beendet.

/giveaway pause

Pausiert den Timer des Giveaways.

/giveaway resume

Setzt den Timer des Giveaways fort.

/giveaway reroll

Lost die Gewinner des Giveaways neu aus.

/giveaway extend

Verlängert die Dauer des Giveaways. Argumente:
dauer: Zeit um die das Giveaway verlängert werden soll. Kann in s|m|h|d angegeben werden.

/giveaway delete

Löscht das gefilterte Giveaway aus der Datenbank. Argumente(Mindestens eins erforderlich):
titel: Titel des Giveaways.
older: Wie alt das Giveaway sein soll um gelöscht zu werden. Kann in s|m|h|d angegeben werden.

Tickets

/ticketmessage

Erstellt die Ticket-Erstellungs-Nachricht im Tickets Kanal. Muss nur ausgeführt werden wenn die Ticket-Erstellungs-Nachricht gelöscht wurde/nicht mehr exsistiert.
